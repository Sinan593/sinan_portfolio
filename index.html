Secure-Chain

Secure-Chain is a web-3 based application which targets customers who prioritize more on privacy.The application is a decentralized file sharing as well as file storing application.The application provides the users to share any type of file and store it in the application for unlimited period of time with a guarantee of the file not getting hacked.

Medical Robo

Medical Robo is a prototype model comprising hardware as well as software to deliver quick response to the the doctor about the patient's health status from the spontaneously spreading diseases.The sensors in the robo takes input from the patient and stores it in the database.The data in the database is then taken and processed by the ML model which identifies whether the patient is infected with a spontaneous disease or not.If infected,the model will also identify it as well.
